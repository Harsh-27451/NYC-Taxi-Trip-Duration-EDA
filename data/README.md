
#  Exploratory Data Analysis of NYC Taxi Trips - Trip Duration

## Introduction

Conducted exploratory data analysis on 729,000+ New York City taxi trip data records.

## Data Source

* NYC Taxi Trip Duration Dataset
* 729K+ rows

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Observations

* Single rider taxis are the most common mode of taxi travel.
* The median value of trip duration is roughly 11 minutes.
* Average speed of travel is about 12 km/hr.
* Data cleaning helped improve the analysis process.
